# 授信额度核定标准

## 一、额度核定原则

1. **综合评分为主**：额度初步测算以综合评分为核心依据
2. **营收规模上限**：企业年营收规模决定授信上限阈值
3. **行业差异化**：不同行业适用不同的额度系数
4. **担保方式调节**：担保方式影响最终额度核定

## 三、担保方式建议规则

| 综合评分 | 建议担保方式 |
|----------|--------------|
| 80-100分 | 信用授信（纯信用） |
| 60-79分 | 信用授信或保证担保 |
| 40-59分 | 保证担保或抵押担保 |
| < 40分 | 抵押担保或拒绝 |

---

## 四、授信期限建议

| 风险等级 | 建议授信期限 |
|----------|--------------|
| 低风险 | 24个月 |
| 中低风险 | 12个月 |
| 中高风险 | 6-12个月 |
| 高风险 | 不予授信 |

---

## 五、特殊情况

### 5.1 新成立企业（< 2年）
- 额度上限不超过500万
- 原则上要求抵押担保
- 期限不超过12个月

### 5.2 高新技量术企业
- 额度系数可上浮10%
- 知识产权可作为辅助担保物

### 5.3 链属企业（核心企业上下游）
- 参照核心企业信用评级
- 可采用供应链票据等方式

## 来源依据
本标准依据金融机构流动资金贷款管理办法及行业授信政策制定。
