# 风控禁入规则清单

## 规则说明
以下规则为「一票否决项」，命中任意一条即禁止准入，不进入评分打分环节。

---

## 一、企业主体资格

| 规则ID | 规则描述 | 校验逻辑 |
|--------|----------|----------|
| R001 | 企业已注销 | 经营状态 ≠ 存续 |
| R002 | 企业已被吊销 | 经营状态 = 吊销 |
| R003 |企业经营异常且未移出 | 经营异常标记 = 是，且未移出 |
| R004 | 注册年限不足1年 | 成立年限 < 1年 |

---

## 二、法人及实控人风险

| 规则ID | 规则描述 | 校验逻辑 |
|--------|----------|----------|
| R005 | 法人失信被执行人 | 法人是否失信 = 是 |
| R006 | 法人被限制高消费 | 法人是否限高 = 是 |
| R007 | 实控人失信被执行人 | 实控人是否失信 = 是 |
| R008 | 实控人被限制高消费 | 实控人是否限高 = 是 |

---

## 三、司法风险

| 规则ID | 规则描述 | 校验逻辑 |
|--------|----------|----------|
| R009 | 存在大额被执行标的 | 被执行标的金额 > 500万元 |
| R010 | 涉及借贷纠纷涉诉 | 借贷纠纷涉诉数量 ≥ 3笔 |
| R011 | 存在失信被执行记录 | 失信记录数量 ≥ 1条 |
| R012 | 股权被冻结 | 股权冻结标记 = 是 |

---

## 四、监管处罚

| 规则ID | 规则描述 | 校验逻辑 |
|--------|----------|----------|
| R013 | 税务非正常户 | 税务状态 = 非正常 |
| R014 | 近2年连续行政处罚 | 行政处罚次数 ≥ 2次（近2年） |
| R015 | 环保处罚 | 环保处罚标记 = 是 |
| R016 | 安监处罚 | 安监处罚标记 = 是 |

---

## 五、空壳公司识别

| 规则ID | 规则描述 | 校验逻辑 |
|--------|----------|----------|
| R017 | 参保人数为0 | 参保人数 = 0 |
| R018 | 无实际经营地址 | 经营地址标记 = 空壳风险 |
| R019 | 无知识产权 | 专利+软著数量 = 0 |
| R020 | 无招投标记录 | 招投标中标次数 = 0（注册2年以上） |

---

## 规则优先级
1. 校验按顺序执行，遇到第一条命中即返回「拒绝准入」
2. 每次校验需记录：规则ID、规则描述、命中状态（是/否）、校验时间

---

## 来源依据
本清单依据金融机构通用风控标准制定，适用于贷前准入审核。
