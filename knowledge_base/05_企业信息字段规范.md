# 企业信息采集字段规范

## 一、工商基础信息

| 字段名 | 说明 | 备注 |
|--------|------|------|
| company_name | 企业全称 | 必填 |
| unified_credit_code | 统一社会信用代码 | 必填，18位 |
| legal_person | 法人代表 | 必填 |
| establish_date | 成立日期 | YYYY-MM-DD格式 |
| business_status | 经营状态 | 存续/注销/吊销/其他 |
| registered_capital | 注册资本 | 万元 |
| paid_capital | 实缴资本 | 万元 |
| business_scope | 经营范围 | 文本 |
| address | 注册地址 | 省市区详细地址 |
| employee_count | 参保人数 | 整数 |
| branch_count | 分支机构数量 | 整数 |

---

## 二、司法风险信息

| 字段名 | 说明 | 备注 |
|--------|------|------|
| is_xinping | 是否失信被执行 | 是/否 |
| is_limit_high_consume | 是否限制高消费 | 是/否 |
| execution_amount | 被执行标的金额 | 万元 |
| lawsuit_count | 涉诉数量 | 合同纠纷/借贷纠纷分开统计 |
| is_stock_frozen | 股权是否冻结 | 是/否 |
| pledge_count | 股权质押数量 | 整数 |
| mortgage_count | 动产抵押数量 | 整数 |

---

## 三、监管处罚信息

| 字段名 | 说明 | 备注 |
|--------|------|------|
| is_abnormal | 是否经营异常 | 是/否 |
| abnormal_reason | 异常原因 | 文本 |
| is_tax_abnormal | 税务是否非正常 | 是/否 |
| penalty_count | 行政处罚次数 | 近2年 |
| penalty_amount | 行政处罚金额 | 万元 |
| is_env_penalty | 是否有环保处罚 | 是/否 |
| is_safety_penalty | 是否有安监处罚 | 是/否 |

---

---

## 数据来源说明
所有字段数据来源于公开企业公示信息渠道，包括但不限于：
- 国家企业信用信息公示系统
- 中国裁判文书网
- 全国法院失信被执行人信息查询
- 国家知识产权局
- 各地行政处罚信息公示平台

本项目使用模拟数据进行演示，数据结构与真实采集保持一致。
