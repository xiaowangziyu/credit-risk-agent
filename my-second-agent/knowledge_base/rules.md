# 企业授信禁入规则

## 一、硬性禁入标准（一票否决）

以下任意一项命中，企业直接判定为**禁入**，不得进入评分环节：

1. **经营状态异常**
   - 企业状态为「吊销」「注销」「停业」「清算中」
   - 近3年内被列入经营异常名录且未移出

2. **严重违法违规**
   - 近2年内存在重大行政处罚记录
   - 被列入失信被执行人名单
   - 存在重大税务违法记录

3. **存续期过短**
   - 企业注册成立不满1年（<12个月）

4. **注册资本过低**
   - 注册资本低于100万元人民币（特殊行业可放宽至50万）

5. **行业禁入**
   - 高污染、高耗能且未通过环保评审的企业
   - 涉及虚拟货币、非法金融活动的企业

## 二、高风险警示标准（需额外尽调）

以下情况虽不直接禁入，但需标记为**高风险**，需补充额外材料：

1. **财务风险**
   - 连续2年亏损
   - 资产负债率超过80%
   - 流动比率低于0.8

2. **经营风险**
   - 主营业务发生重大变更未满1年
   - 核心管理层变动频繁（近1年变动超过30%）

3. **行业风险**
   - 属于国家政策限制类行业
   - 行业周期性强且处于下行周期

## 三、评分准入门槛

通过禁入校验后，需满足以下最低评分标准：

- **综合评分 ≥ 60分**方可进入授信建议环节
- **评分 < 60分**：建议拒绝授信或仅提供低额度、短期限的担保类授信
