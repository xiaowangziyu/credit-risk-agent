# 企业信用评分维度参考指南

## 一、核心评分维度（建议权重分配）

### 1. 企业基本资质（15-20分）
- 注册年限：越长越稳定（满3年基准分，每增加2年加分）
- 注册资本：反映企业实力
- 经营状态：是否存续、有无异常记录
- 法定代表人信用：个人征信状况

### 2. 经营稳定性（20-25分）
- 年营业收入规模及增长率
- 利润水平及盈利趋势
- 员工人数及稳定性
- 核心业务连续性

### 3. 财务健康度（20-25分）
- 资产负债率
- 流动比率/速动比率
- 应收账款周转情况
- 现金流状况

### 4. 行业与市场地位（10-15分）
- 所属行业景气度
- 企业在行业中的地位
- 市场份额变化趋势
- 政策支持力度

### 5. 信用记录与合规性（15-20分）
- 银行贷款还款记录
- 有无失信记录
- 行政处罚情况
- 税务合规情况

### 6. 担保与增信能力（5-10分，可选）
- 可抵质押资产情况
- 第三方担保能力
- 关联企业实力

## 二、评分等级划分

| 综合评分 | 风险等级 | 建议授信策略 |
|---------|---------|-------------|
| 85-100分 | 低风险 | 可给予较高额度授信，利率可优惠 |
| 70-84分 | 中低风险 | 可给予正常额度授信，标准利率 |
| 60-69分 | 中风险 | 需担保或抵押，额度适度控制 |
| 50-59分 | 中高风险 | 建议仅低额度短期授信，附加增信 |
| <50分 | 高风险 | 建议拒绝授信 |

## 三、扣分规则说明

- 每项维度按满分逐项评估
- 存在明确负面信息的按程度扣分
- 存在正面加分项的可在基准分上适当上浮
- 信息缺失的维度按保守评估（给予基准分的60%-70%）
