# 授信额度与期限建议参考

## 一、授信额度测算方法

### 1. 基础额度计算公式（推荐优先使用）

```
基础额度 = MIN(
    年营业收入 × 收入系数,
    注册资本 × 2,
    净资产 × 70%
)
```

**收入系数对照表**：

| 综合评分 | 收入系数 | 说明 |
|---------|---------|------|
| 85-100分 | 25% | 低风险，可给予较高额度 |
| 70-84分 | 17% | 中低风险，正常额度 |
| 60-69分 | 12% | 中风险，适度控制 |
| 50-59分 | 7% | 中高风险，低额度 |
| <50分 | 0% | 高风险，不建议授信 |

### 2. 综合额度计算

```
最终额度 = 基础额度 × 行业系数 × 担保系数
```

**行业系数**：

| 行业类别 | 调整系数 |
|---------|---------|
| 科技服务业 | 1.2 |
| 医疗器械 | 1.1 |
| 电子制造 | 1.0 |
| 装备制造 | 1.0 |
| 贸易流通 | 0.95 |
| 物流运输 | 0.95 |
| 食品生产 | 0.95 |
| 建筑工程 | 0.85 |
| 金融服务 | 0.8 |

**担保系数**：

| 担保方式 | 系数 |
|---------|------|
| 纯信用 | 0.8 |
| 第三方担保 | 1.0 |
| 房产抵押 | 1.2 |
| 设备抵押 | 1.1 |
| 应收账款质押 | 1.0 |
| 多重担保组合 | 1.3 |

### 3. 额度限制

- **单户上限**：不得超过净资产的70%
- **最低额度**：不低于10万元
- **关联企业**：总额不得超过单户限额的1.5倍

### 4. 快速估算（简化版）

对于初筛场景，可使用简化公式：

```
估算额度 ≈ MIN(年营收 × 15%, 注册资本 × 1.5) × 行业系数
```

## 二、授信期限建议

| 综合评分 | 建议授信期限 | 说明 |
|---------|-------------|------|
| 85-100分 | 12-24个月 | 可给予长期限 |
| 70-84分 | 6-12个月 | 标准期限 |
| 60-69分 | 3-6个月 | 适度缩短 |
| 50-59分 | 3个月以内 | 短期观察 |
| <50分 | 不建议授信 | - |

**行业特殊期限**：
- 建筑工程类：建议6-12个月（与工程周期匹配）
- 设备采购类：建议12-24个月
- 流动资金类：建议6-12个月

## 三、授信类型建议

| 授信类型 | 适用场景 | 建议期限 |
|---------|---------|---------|
| 流动资金授信 | 日常经营周转、货物采购备货、物流运费支付 | 6-12个月 |
| 固定资产授信 | 设备购置更新、厂房建设 | 12-24个月 |
| 仓单质押授信 | 有稳定库存存货的企业 | 3-6个月 |
| 应收账款保理授信 | 应收账款占比较高、下游客户优质 | 6-12个月 |
| 投标保函授信 | 参与招投标项目的企业 | 按项目周期 |
| 纯信用授信 | 评分较高（≥85分）、历史合作良好的企业 | 6-12个月 |

**授信类型判断优先级**：
1. 查看企业经营范围和资金实际用途
2. 根据行业特征匹配授信类型
3. 优先推荐流动资金授信（灵活性高）

## 四、资金用途建议

- **日常经营周转**：适用于所有企业，最常见用途
- **货物采购备货**：适用于贸易流通、生产制造企业
- **物流运费支付**：适用于物流运输、电商企业
- **设备购置更新**：适用于制造业、装备更新需求
- **项目履约垫资**：适用于建筑工程、项目型企业

**用途合规要求**：
- 不得用于股权投资、房地产投资
- 不得用于民间借贷、非法金融活动
- 应保留资金流向证明材料

## 五、利率建议

| 风险等级 | 建议年化利率范围 |
|---------|----------------|
| 低风险（85-100分）| 6%-9% |
| 中低风险（70-84分）| 9%-12% |
| 中风险（60-69分）| 12%-15% |
| 中高风险（50-59分）| 15%-18% |
| 高风险（<50分）| 不建议授信 |

**利率计算说明**：
- 实际利率不得超过监管上限（参考：同期LPR×4）
- 可根据担保方式适当调整
- 优质客户可享受利率优惠
